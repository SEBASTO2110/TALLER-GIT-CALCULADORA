## Descripcion 
1. Fibonacci
- Genera la serie de Fibonacci hasta una cantidad de términos indicada por el usuario.

2. Número capicúa
- Verifica si un número se lee igual de izquierda a derecha que de derecha a izquierda.

3. Número perfecto
- Determina si un número es perfecto, es decir, si la suma de sus divisores propios es igual al número.

4. Números primos en un rango
- Muestra todos los números primos que se encuentran entre un valor inicial y un valor final ingresados por el usuario.

5. Verificar si un número es primo
- Determina si un número ingresado por el usuario es primo o no.

6. Factorial
- Calcula el factorial de un número entero positivo ingresado por el usuario.

7. Máximo Común Divisor (MCD)
- Calcula el mayor número que divide exactamente a dos números ingresados por el usuario.
